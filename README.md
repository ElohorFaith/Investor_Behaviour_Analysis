# Investor_Behaviour_Analysis
This project investigates investor behavior using survey data to understand how factors such as demographics, financial goals, employment status, and investment duration affect financial decision-making.

# Dataset Description
1. Gender
2. Age
3. Working professional
4. Annual income
5. Mode of investment
6. Investment per month
7. Motivation cause
8. Resources used
9. Goal for investment
10. Duration to save(in years)

# Part 1: Exploratory Data Analysis (Excel)
The initial analysis was performed in Excel using Pivot tables and charts.
Key Excel analysis performed:
1. Demographic distribution
2. Investment behaviour analysis
3. Age vs investment goal analysis
4. Goal vs investment duration

# Key Excel Insight
1. The dataset is dominated by young investors that prioritize wealth creation and personal savings.
2. As people become older, their investment objectives increasingly change to include formal planning for the future and retirement.
3. The average monthly investment contribution is only slightly different depending on one's working status.
4. Long-term duration bands are closely linked to retirement-oriented goals.

# Part 2: PowerBI Dashboard
An interactive dashboard was created to turn the exploratory study into a visual report suited to executives.

# Dashboard Features:
1. KPI Cards which includes: Total Respondents, Average Monthly Investments, Total Monthly Investments.
2. Gender Distribution Donut chart
3. Age Group distribution.
4. Mode of Investment analysis
5. Average monthly investment by working status
6. 100% stacked column to show investment goals by Age Group.
7. Investment goal by duration band
8. Interactive slicers for dynamic filtering.

# Data Modeliing and DAX Enhancement
1. Created Age group segmentation using DAX
2. Created Duration band grouping using DAX: Short Term, Medium Term and Long Term.
3. Created custom measures for KPI calculations.
4. Applied proper currency formatting and sorting logic.

# Key Business Insights
1. Younger investors place a higher priority on wealth accumulation and liquidity.
2. Investors in their middle years start to include retirement and family-related financial planning.
3. Duration of investment is strongly aligned with Financial Objectives.
4. Monthly investing behavior is not significantly influenced by working status alone.

# Business Recommendations
1. Financial institutions need to create investment products that are divided into different life stages.
2. Middle-aged investors should be the target market for retirement-focused products.
3. Young professionals are a high-opportunity market for long-term wealth products.
4. Duration-based planning may be impacted by investment education programs.
